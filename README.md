# Birdymap - a Mashup Web Application

The primary goal of this project is to create an application dedicated to bird observation. 

This application aims to grant bird enthusiasts worldwide easy access to data regarding recently sighted bird species. The process involves users selecting a particular bird species, which is linked to a specific country on an interactive map. 

Once a selection is made, users gain the ability to explore a curated collection of pertinent YouTube videos and Flickr images associated with the chosen bird species. 

This interactive platform allows users to visually and audibly acquaint themselves with the appearance and sounds of the selected bird. 

To facilitate this process, the application is deployed on a public cloud Virtual Machine, utilizing the AWS EC2 service. This deployment is efficiently managed within a Docker container for streamlined hosting.

# APIs used
- Leaflet open-source JavaScript library for interactive maps (v. 1.8.0)
- MapTiler (v. Winter)
- eBird API (v. 2.0)
- YouTube Data API (v. 3.0)
- Flickr (v. 3.0)
